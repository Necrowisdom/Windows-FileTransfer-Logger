# Windows-FileTransfer-Logger
A GUI-based Windows utility for moving folders with automated audit logging, real-time progress tracking, and standalone executable support.

### Why use FileFort-LogTrans?
- **Zero Configuration:** Download the `.exe` and start moving files.
- **Detailed Logs:** Every action is saved with timestamps in `dosya_hareketleri.log`.
- **Safe:** Multi-threaded processing prevents the app from freezing during large transfers.
